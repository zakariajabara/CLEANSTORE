This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.3.23.

To Stat the Application:

   You cann access it on the URL: http;//localhost:3500/products)

in the an other Terminal RUN
   ng serve --port 3000 --open ( you can choose an other port if desired)
